# Digital Banking Transaction Platform

A backend banking application built using Java, Spring Boot, and SQL Server that supports secure account management, fund transfers, and transaction processing using REST APIs and ACID-compliant database operations.

---

# Features

- Create bank accounts
- View all accounts
- Transfer funds between accounts
- RESTful API architecture
- Layered Spring Boot architecture
- SQL Server database integration
- Transaction-safe operations
- Maven-based project structure

---

# Tech Stack

- Java 17
- Spring Boot
- Spring Data JPA
- SQL Server
- Maven
- Lombok
- REST APIs
